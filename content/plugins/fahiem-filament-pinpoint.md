Filament Pinpoint is a Google Maps location picker component for Filament 4.

   ## Features

   - Search location using Google Places Autocomplete
   - Click to set marker
   - Draggable marker
   - Get current device location
   - Reverse geocoding - auto-fill address fields
   - Fully configurable
